## compile and run locally

- `git clone --recursive https://github.com/Lorthiz/kaitai_struct_webide`
- `npm install`
- `npm run generate-formats`
- `npm run parcel-run`
- Go to [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
